# Components

## FeedGrid
contains all blocks

## Block
props:
- type (string) (youtube, hn, etc)
  - select classNames by type
- data (array|object)

type prop decides colors and which list item component is used for data

## YoutubeListItem
props:
- items (array) list of items to show

## HNListItem
props:
